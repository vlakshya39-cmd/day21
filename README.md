# Day 21 – Digital Privacy Dashboard

## Objective

Build an interactive Digital Privacy Dashboard using Claude AI to visualize digital footprint exposure, privacy risks, and data collection patterns based on a sample user dataset.

---

## Tools Used

* Claude AI
* HTML
* CSS
* JavaScript
* GitHub

---

## Project Overview

For Day 21, I used Claude AI to generate a fully interactive Digital Privacy Dashboard. The dashboard analyzes a sample digital footprint and presents privacy-related insights through scores, rankings, heatmaps, and risk assessments.

The dashboard demonstrates how AI can transform structured data into a professional cybersecurity-style analytics interface.

---

## Features Included

* Digital Footprint Score
* Privacy Score
* Exposure Heatmap
* Company Exposure Ranking
* Data Collection Matrix
* Risk Radar
* Digital Twin Profile
* Most Valuable Data Assets
* WOW Insights
* Privacy Improvement Plan
* Final Verdict

---

## Key Learnings

* Learned the concept of digital footprints and online data exposure.
* Understood how different services contribute to privacy risks.
* Explored data visualization techniques used in cybersecurity dashboards.
* Observed how AI can generate complete front-end applications using HTML, CSS, and JavaScript.
* Gained insights into privacy awareness and digital ecosystem concentration.
